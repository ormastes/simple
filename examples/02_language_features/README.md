# Language Features Examples

Demonstrations of Simple language syntax and features.

## Categories

### Blocks (`blocks/`)
Custom block syntax and user-defined blocks

### Polymorphism (`polymorphism/`)
Static polymorphism and type system features

### AOP (`aop/`)
Aspect-oriented programming demonstrations

### Syntax (`syntax/`)
Parser syntax examples (async, attributes, spawn)

## Top-level Examples

- **execution_context.spl** - Execution context types and switching
- **lean_blocks.spl** - Lean functional-style blocks
