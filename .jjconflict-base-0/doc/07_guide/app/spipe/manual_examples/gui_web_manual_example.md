## Creating a Note

1. Click **New Note** in the toolbar.
2. Type a title in the title field.
3. Click into the body and type your note text.

The note appears at the top of the list as you type, and the editor keeps your
place:

![Creating a note](captures/notes_create.png)

Checked components: `note_list` (new row inserted at top), `editor_pane`
(title + body fields), `save_indicator`. Masked: cursor caret, `updated_at`
timestamp.

Your note saves automatically a moment after you stop typing вҖ” watch the
**Saved** label appear next to the title.

---

## Searching Notes

1. Click the search field above the note list.
2. Type part of a title or body word.

The list filters as you type, and matching text is highlighted:

![Searching notes](captures/notes_search.png)

Checked components: `search_box`, `note_list` (filtered rows + highlight
spans). Masked: `updated_at` timestamps on remaining rows.

Clear the search field to see every note again.

## Edge Case: Unsaved Changes

**If you start editing a note, then click another note before it saves**, a
warning dialog appears so you don't lose text:

![Unsaved changes warning](captures/notes_unsaved_warning.png)

Checked components: `unsaved_dialog` (title, message, `discard_button`,
`cancel_button`). Masked: none.

1. Click **Keep Editing** to stay on the current note and let it save.
2. Or click **Discard Changes** to switch notes anyway and lose the edit.

## Troubleshooting

| Symptom | Cause | Fix |
|---|---|---|
| **Saved** label never appears | Network request to save endpoint failed | Check your connection; the app retries automatically |
| Search box shows no results for a word you can see in a note | Search only matches title + first 200 characters of body | Scroll to the note manually, or search a word from its title |
| Unsaved-changes dialog appears when switching notes you didn't edit | A prior edit was still pending save | Click **Keep Editing** and wait for **Saved** before switching |
| Editor pane is blank after creating a note | Browser window narrower than 480px collapses the editor pane | Widen the window or rotate to landscape on mobile |
