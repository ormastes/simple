# theme_html — Default Classless CSS Theme

## Entries

- **simple_default.css** — Original classless CSS theme covering ~50 HTML elements (no class selectors). Sakura-inspired philosophy with CSS variables, flexbox nav, dark mode support via prefers-color-scheme. Unminified size: ~3.2 KB.

- **theme_showcase.html** — Comprehensive HTML5 showcase exercising all major semantic elements: typography (h1–h6, p, strong, em, code, pre, blockquote, hr), forms (input types, button, select, textarea, checkbox, radio, label, fieldset, legend), structure (header, nav, main, section, article, aside, footer), lists (ul, ol, li), tables (table, thead, tbody, tr, th, td, caption), interactive (details, summary, progress, dialog reference), and images.
