# T32 Agent - TRACE32 MCP & CMM Plugin Setup

## MCP Servers

This agent requires the following MCP servers (not loaded by default — enable when spawning T32 tasks):

```json
{
  "t32-mcp": {
    "command": "bin/t32_mcp_server",
    "args": [],
    "cwd": "."
  },
  "t32-lsp-mcp": {
    "command": "bin/t32_lsp_mcp_server",
    "args": [],
    "cwd": "."
  }
}
```

**Enable before use:** Add these to `.mcp.json` or run `claude mcp add` when working on T32 tasks.

**Use when:** Installing T32 MCP server, setting up CMM LSP plugin, configuring TRACE32 integration.
**Skills:** `/t32`

## T32 MCP Server Installation

### Claude Code CLI (`claude mcp add`)

```bash
# T32 MCP — hardware debug session control (36 tools)
claude mcp add t32-mcp -- \
  /absolute/path/to/simple/bin/release/simple \
  /absolute/path/to/simple/examples/10_tooling/trace32_tools/t32_mcp/main.spl

# T32 LSP MCP — CMM language intelligence (6 tools)
claude mcp add t32-lsp-mcp -- \
  /absolute/path/to/simple/bin/release/simple \
  /absolute/path/to/simple/examples/10_tooling/trace32_tools/t32_lsp_mcp/main.spl
```

Replace `/absolute/path/to/simple` with the actual project root.

### Manual `.mcp.json` (project root)

```json
{
  "mcpServers": {
    "t32-mcp": {
      "command": "bin/release/simple",
      "args": ["examples/10_tooling/trace32_tools/t32_mcp/main.spl"],
      "cwd": "/absolute/path/to/simple"
    },
    "t32-lsp-mcp": {
      "command": "bin/release/simple",
      "args": ["examples/10_tooling/trace32_tools/t32_lsp_mcp/main.spl"],
      "cwd": "/absolute/path/to/simple"
    }
  }
}
```

### Environment Variables (optional overrides)

| Variable | Default | Purpose |
|----------|---------|---------|
| `T32_DEFAULT_HOST` | localhost | T32 host |
| `T32_DEFAULT_PORT` | 20000 | T32 port |
| `T32_RCL_PORT` | 20000 | RCL API port |
| `T32_RCL_HOST` | localhost | RCL API host |
| `T32_BACKEND_PREFERENCE` | (auto) | `t32rem`, `ctypes`, or `python_rcl` |
| `T32_PYTHON` | python3 | Python binary for python_rcl backend |
| `T32_API_LIB` | | Path to T32 API shared library |
| `T32_PYTHON_BRIDGE` | | Path to Python bridge script |

Config file: `config/t32/t32_mcp.sdn` (loaded automatically, env vars override).

## CMM LSP Plugin Installation

### Claude Plugin (marketplace)

Plugin location: `tools/claude-plugin/marketplace/plugins/cmm-lsp/`

```bash
# Install CMM LSP plugin for .cmm file support
cp -r tools/claude-plugin/marketplace/plugins/cmm-lsp/.claude-plugin /path/to/target/
cp tools/claude-plugin/marketplace/plugins/cmm-lsp/.lsp.json /path/to/target/
```

### LSP Configuration (`.lsp.json`)

```json
{
  "languages": [
    {
      "fileExtensions": [".cmm"],
      "languageId": "cmm",
      "command": ["bin/release/simple", "examples/10_tooling/trace32_tools/cmm_lsp/mod.spl", "--lsp"],
      "initializationOptions": {}
    }
  ]
}
```
