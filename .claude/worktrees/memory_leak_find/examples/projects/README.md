# Full-Scale Projects

Complete applications and larger demonstrations.

## MedGemma Korean (`medgemma_korean/`)

Korean medical language model based on Google's MedGemma.

### Structure
- **src/** - Training implementation (LoRA, validation, multi-phase training)
- **config/** - Training configurations
- **models/** - Model architectures
- **research/** - Research notes and experiments

### Documentation
See `medgemma_korean/README.md` for details.

## Adding Projects

Full-scale projects should:
1. Have their own directory with README.md
2. Include complete source structure (src/, config/, etc.)
3. Be self-contained and runnable
4. Demonstrate real-world usage patterns
