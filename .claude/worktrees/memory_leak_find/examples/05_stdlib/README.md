# Standard Library Examples

Demonstrations of stdlib features and utilities.

## Examples

- **platform_library.spl** - Platform abstraction layer usage
  - Cross-platform path handling
  - OS detection
  - Newline normalization
  - Endianness handling

## Standard Library Modules

- **std.string** - String operations
- **std.array** - Array utilities
- **std.math** - Math functions
- **std.path** - Path manipulation
- **std.platform** - Platform abstraction
- **std.convert** - Type conversions
- **std.spec** - SSpec testing framework
