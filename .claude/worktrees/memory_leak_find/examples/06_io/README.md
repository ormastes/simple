# I/O Examples

File I/O, networking, graphics, and multimedia.

## Categories

### File (`file/`)
- **file_staging.spl** - Memory-mapped file staging (mold-inspired)
- **file_staging_parallel.spl** - Parallel file processing with shared memory

### Network (`network/`)
- **http_client.spl** - HTTP client example

### Graphics (`graphics/`)
- **graphics2d.spl** - 2D graphics rendering
- **window_creation.spl** - Window management

### Multimedia (`multimedia/`)
- **audio_playback.spl** - Audio playback example
- **gamepad_input.spl** - Gamepad input handling
