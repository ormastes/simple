FreeBSD Simple Compiler
=======================

Binary: bin/freebsd/simple
Type: seed_cpp transpiler (Simple → C++ → Native)
Architecture: x86-64 FreeBSD 14.1
Size: 79KB
Built: $(date)
Source: build/freebsd/seed_cpp (cross-compiled from Linux)

Features:
- Core Simple subset (val/var, fn, struct, enum, impl, control flow)
- NOT supported: generics, lambdas, comprehensions, async, classes

Usage in FreeBSD VM:
  ./simple input.spl > output.cpp
  clang++ -std=c++20 -o output output.cpp runtime.c -Iruntime

Full compiler build (in progress):
- Requires native C backend or LLVM backend
- Current limitation: seed_cpp cannot handle full 439-file compiler
