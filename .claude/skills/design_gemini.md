# Design Skill (Gemini) -- Step 3: TUI/GUI Design Generation

**Pipeline:** Step 3 of 5 (after research_codex, before design_codex)

## Prerequisites
- Chosen requirements from `doc/02_requirements/feature/` and `doc/02_requirements/nfr/`

## Phase 1: TUI Design

1. Generate TUI mockup examples in markdown
2. Create base UI component list for terminal interface
3. Present TUI list to user for confirmation
4. Iterate based on user feedback

### Output
- `doc/05_design/<feature>_tui.md`

## Phase 2: GUI Design

1. Generate GUI mockup examples with web-based patterns
2. Create web base GUI component list
3. Present web GUI list to user for confirmation
4. Iterate based on user feedback

### Output
- `doc/05_design/<feature>_gui.md`

## Handoff
Pass UI designs to `/design_codex` (Step 4).
