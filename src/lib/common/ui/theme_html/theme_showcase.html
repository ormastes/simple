Click to expand: What is Simple?

Simple is a self-hosted compiler and systems programming language. It features:

  • No garbage collection
  • Memory safety through composition and traits
  • Runs on Linux, macOS, FreeBSD, Windows
  • Bootstrap from Rust to self-hosted Simple
Click to expand: Why classless CSS?

Classless CSS themes provide:

  • Zero markup coupling (no class names to remember)
  • Semantic HTML styling by default
  • Smaller CSS payload
  • Lower barrier to entry for LLM-generated content
