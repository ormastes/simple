# markdown-oxide-lsp

Markdown Oxide language server plugin for Claude Code.

Provides completions, references, links, hover, and diagnostics for Markdown files (Obsidian-compatible).

## Prerequisites

Install markdown-oxide:

```bash
cargo install --locked --git https://github.com/Feel-ix-343/markdown-oxide.git markdown-oxide
```

## Vault

The `doc/` folder is an Obsidian vault (has `.obsidian/` directory).
