# examples/ Manifest

Example projects demonstrating Simple language features, applications, and integrations.

## Allowed Entries

| Entry | Description |
|---|---|
| `01_getting_started` | Hello world and introductory examples |
| `02_language_features` | Language features: polymorphism, syntax, blocks, AOP, BDD |
| `03_concurrency` | Concurrency patterns and async examples |
| `04_data_formats` | Data format handling (JSON, CSV, XML, SDN) |
| `05_stdlib` | Standard library usage examples (SPipe) |
| `06_io` | I/O examples: UI, networking, web servers, TLS, terminal |
| `07_ml` | Machine learning, deep learning, LLM tools |
| `08_gpu` | GPU computing: CUDA examples and patterns |
| `09_embedded` | Embedded/OS: SimpleOS, FPGA, QEMU, baremetal, RISC-V |
| `10_tooling` | Developer tools: MCP servers, debuggers, build systems, IDE |
| `11_advanced` | Advanced projects: browser engine, game2d, databases |
| `FILE.md` | This manifest |
