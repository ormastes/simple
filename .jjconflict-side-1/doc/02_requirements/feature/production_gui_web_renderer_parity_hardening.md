# Production GUI Web Renderer Parity Hardening Feature Requirements

This is the canonical SPipe feature-slot copy for the existing selected UI
requirements in `doc/02_requirements/ui/misc/production_gui_web_renderer_parity_hardening.md`.

The active selected requirements are REQ-001 through REQ-025 in that UI
requirements document. They cover marker-free generated GUI HTML, generic
Simple Web layout rendering, exact software/CPU SIMD/Metal parity, live
Electron/Web reference capture, viewport matrix evidence, CSS/layout manifest
coverage, and zero-mismatch/tolerance-free reporting.

For production-browser hardening work outside renderer parity, use
`.spipe/simple_web_browser_production_hardening/state.md` and
`doc/03_plan/agent_tasks/simple_web_browser_production_hardening.md`.
