# Simple 3D Graph IR NFR

| Requirement | Statement |
|-------------|-----------|
| NFR-3D-GRAPH-001 | Optimization must be deterministic for identical input graphs. |
| NFR-3D-GRAPH-002 | The initial implementation must be pure Simple library code and avoid new runtime, C, Rust, CUDA, Vulkan, or WebGPU dependencies. |
| NFR-3D-GRAPH-003 | Invalid mesh or pipeline handles must not emit draw nodes. |
