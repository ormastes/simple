Simple Runtime Limitations Discovered
--------------------------------------
1. Enum variant pattern matching: only the first pattern variable is
   bound. For multi-field variants, the first binding receives the
   entire variant data as a tuple. Use d[0], d[1], d[2] indexing.

2. Pass-by-value semantics: arrays, classes, and structs are all
   passed by value to functions. Mutations via .push() or me methods
   don't persist to the caller. Workaround: use module-level vars
   (works for imported modules) or return-value pattern.

3. Module-level mutation only works for IMPORTED modules: functions
   defined in the same file as module-level vars cannot modify them.
   Functions imported via `use module.{func}` CAN modify their own
   module's state.
