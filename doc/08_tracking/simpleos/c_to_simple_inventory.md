## 6. Port priority summary (input to AC-2 sequencing)

**P0 (critical kernel + libc base; ~10 files, ~5 KLoC):**
- `runtime_minimal.c` (kernel-side baremetal runtime — CRITICAL)
- `simpleos_libc.c`, `simpleos_fs.c`, `simpleos_stdlib_ext.c`, `simpleos_string_ext.c`, `simpleos_time.c`, `simpleos_alloc.c`, `simpleos_pthread.c`, `simpleos_process.c`, `simpleos_cxxabi.c`
- Per-arch `crt0.S` + `setjmp.S` — these MUST be lifted into the HAL (`os.hal.<arch>.cstart`) and given armv7/aarch64/rv32imac/rv64gc/i686 implementations to satisfy AC-3+AC-4.

**P1 (hosted runtime + secondary libc; ~15 files, ~8 KLoC):**
- `runtime.c`, `runtime_thread.c`, `runtime_native.c`, `runtime_memtrack.c`, `async_driver.c`
- `simpleos_epoll.c`, `simpleos_poll.c`, `simpleos_signal.c`, `simpleos_dlmalloc.c`, `simpleos_math_ext.c`, `simpleos_printf_float.c`, etc.

**P2 (host-platform-specific or hosted-only):**
- `platform/async_*.c` (linux/win/mac/fbsd) — port to Simple but per-host conditional, not on SimpleOS-kernel path.
- `runtime_sdl2.c`, `runtime_audio.c`, `runtime_font.c` — hosted-only.

**P3 (headers / interface contracts):** keep as-is.
