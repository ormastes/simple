## Getting Started

Launch the app from a terminal:

```sh
todo
```

You'll see the main screen — a task list on top and an input bar at the bottom:

```text
┌ Todo ────────────────────────────────┐
│                                      │
│          No tasks yet.               │
│          Press a to add one.         │
│                                      │
├──────────────────────────────────────┤
│ Add: _                               │
└──────────────────────────────────────┘
 0 tasks · a:add  d:done  q:quit
```

The bottom line always shows the keys available on the current screen.

## Adding a Task

1. Press `a` to focus the input bar.
2. Type the task name — for example, `buy milk`.
3. Press `Enter`.

The task appears in the list immediately:

```text
┌ Todo ────────────────────────────────┐
│ > [ ] buy milk                       │
│                                      │
│                                      │
├──────────────────────────────────────┤
│ Add: _                               │
└──────────────────────────────────────┘
 1 task · a:add  d:done  q:quit
```

The `>` marker shows the selected task; `[ ]` means it is not done yet.

**If you press Enter on an empty input**, nothing is added and the input bar
shows a hint:

```text
├──────────────────────────────────────┤
│ Add: _  (task name cannot be empty)  │
└──────────────────────────────────────┘
```

---

## Completing a Task

1. Select the task with `↑`/`↓`.
2. Press `d`.

The task is marked done and the counter updates:

```text
┌ Todo ────────────────────────────────┐
│ > [x] buy milk                       │
└──────────────────────────────────────┘
 1 task · 1 done
```

Completed tasks sync to the server automatically. You don't need to do
anything — see *When sync fails* below if you work offline.

**When sync fails** (no network), the task stays done locally and the status
line shows a pending marker until the connection returns:

```text
 1 task · 1 done · ⇅ 1 pending sync
```

Pending changes are retried automatically. You never lose a completed task by
going offline.

## Keyboard Reference

| Key | Action | Where |
|---|---|---|
| `a` | Add a new task | Main screen |
| `Enter` | Confirm task name | Input bar |
| `Esc` | Cancel input | Input bar |
| `↑` / `↓` | Select task | Task list |
| `d` | Mark selected task done | Task list |
| `q` | Quit | Main screen |

## Troubleshooting

| Symptom | Cause | Fix |
|---|---|---|
| `⇅ N pending sync` never clears | Server unreachable | Check network; syncs retry automatically |
| Task not added on Enter | Empty task name | Type a name before pressing Enter |
| Screen looks garbled | Terminal < 40 columns | Resize terminal; minimum is 40×10 |
