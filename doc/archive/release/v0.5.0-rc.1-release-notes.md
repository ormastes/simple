# Simple Language v0.5.0-rc.1 Release Notes

**Release Date:** 2026-02-06  
**Release Type:** Release Candidate  
**Status:** Ready for Testing  
**Tag:** v0.5.0-rc.1

## 🎯 Release Highlights

This release candidate introduces **comprehensive multi-platform bootstrap support**, enabling Simple Language to run seamlessly across 7 different platform combinations.

### Key Features
✅ 7-platform support (Linux, macOS, Windows)  
✅ Automatic platform detection  
✅ GitHub Actions CI/CD pipeline  
✅ 32 comprehensive tests (100% passing)  
✅ Complete documentation suite  

## 🚀 Quick Start

```bash
# Clone or download
git clone https://github.com/ormastes/simple.git
cd simple

# Run on any platform
bin/simple --version
bin/simple your_script.spl
```

## 📦 What's New

### Multi-Platform Bootstrap System
- **7 platforms**: linux-x86_64, linux-arm64, linux-riscv64, macos-x86_64, macos-arm64, windows-x86_64, windows-arm64
- **Smart detection**: Automatic OS and architecture detection
- **CI/CD ready**: GitHub Actions builds all platforms in ~20 minutes

### Performance
- **Binary size**: 31-32 MB (optimized)
- **Startup time**: ~6ms (very fast)
- **Build time**: ~20 min (all platforms, parallel)

### Testing
- **32 tests** covering all functionality
- **100% pass rate**
- Comprehensive verification scripts
