### Quick Install (Linux/macOS)

```bash
curl -fsSL https://raw.githubusercontent.com/simple-lang/simple/main/script/install.sh | sh
```

### Manual Install

```bash
# Download for your platform
wget https://github.com/simple-lang/simple/releases/download/v0.6.0/simple-bootstrap-0.6.0-linux-x86_64.spk

# Extract
tar -xzf simple-bootstrap-0.6.0-linux-x86_64.spk -C ~/.local

# Add to PATH
export PATH="$HOME/.local/bin:$PATH"

# Verify
simple --version
# Output: Simple Language v0.6.0
```

### Windows Install

# Extract to C:\Program Files\Simple
# Add C:\Program Files\Simple\bin to PATH
