# SimpleOS Sysroot Layout

## Purpose

When targeting `x86_64-simpleos` (or sibling architectures such as
`aarch64-simpleos`), cross-compilers must know where to find C headers,
static libraries, and linker scripts for the target. The SimpleOS sysroot at
`build/os/sysroot/` provides exactly that root: every `-I`, `-L`, and `-T`
flag used by the LLVM and Rust cross-toolchains resolves relative to this
directory. Nothing under `build/os/sysroot/` is hand-authored; it is fully
regenerated by `sysroot.shs` from the canonical sources listed below.

## Canonical Layout

| Deliverable | Build path | Source of truth |
|---|---|---|
| Headers | `build/os/sysroot/include/` | `src/os/libc/include/` + `src/os/libc/simpleos_libc.h` |
| Static libc | `build/os/sysroot/lib/libsimpleos_c.a` | `src/os/libc/` Makefile output |
| C runtime object | `build/os/sysroot/lib/crt0.o` | `src/os/libc/simpleos_crt0.o` |
| Userspace LD script | `build/os/sysroot/share/simpleos/simpleos.ld` | Inline script embedded in `sysroot.shs` |

## Userspace vs Kernel Linker Scripts

Two distinct linker scripts exist in the tree and must not be confused:

- **Sysroot script** (`build/os/sysroot/share/simpleos/simpleos.ld`) — used
  when linking userspace applications that run under the SimpleOS kernel.
  Entry point is `_start`; load base is `0x10000000`; no multiboot header.
  This script is embedded verbatim inside `src/os/port/llvm/sysroot.shs` and
  is written into the sysroot at staging time.

- **Kernel script** (`examples/simple_os/arch/x86_64/linker.ld`) — used for
  the baremetal QEMU integration harness. Entry point is `_entry32`; load
  base is `0x100000`; includes a multiboot2 header section. Do not pass this
  script to userspace builds.

## Invocation

```sh
bin/simple run src/os/port/llvm/sysroot.shs
```

Expected runtime: ~30 s on first run (compiles libc from source); ~2 s on
subsequent runs (staging only, sources unchanged).

## Expected Sizes

| Path | Approximate size |
|---|---|
| `include/` (tree) | ~400 KB |
| `lib/libsimpleos_c.a` | ~2–5 MB |
| `lib/crt0.o` | ~3 KB |
| `share/simpleos/simpleos.ld` | ~2 KB |
| **Total sysroot** | **~5 MB** |

## Consumed By

- `scripts/build_llvm_simpleos_cross.sh` — LLVM cross-toolchain build; sets
  `--sysroot=build/os/sysroot`.
- `scripts/build_rust_simpleos_cross.sh` — Rust cross-compile; exports
  `SDKROOT=$SIMPLEOS_SYSROOT`.
- Any `cargo build --target x86_64-unknown-simpleos.json` invocation that
  sets `SDKROOT=$SIMPLEOS_SYSROOT` in the environment.

## Troubleshooting

| Symptom | Fix |
|---|---|
| Missing `libsimpleos_c.a` | `make -C src/os/libc` |
| Missing `crt0.o` | `make -C src/os/libc` |
| Stale or incomplete headers | `rm -rf build/os/sysroot` then re-run `sysroot.shs` |
