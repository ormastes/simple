## Quick Start Guide

### For Contributors

**Choose by Interest:**
- 🔰 **Beginner:** TODO extractor, Hash tool, Snake game
- 📚 **Intermediate:** JSON processor, CSV tool, Calculator
- 🚀 **Advanced:** Web framework, Database, Code editor
- 🧠 **Expert:** Neural network, Game engine, API gateway

**Choose by Time:**
- 📅 **Weekend:** TODO extractor, Code stats, Hash tool
- 📆 **Week:** JSON processor, Test runner, HTTP client
- 🗓️ **Month:** Web framework, Code editor, Database

### Getting Started

```bash
# Clone repository
cd simple/app/

# Pick an application
mkdir my_app
cd my_app

# Create main.spl
cat > main.spl << 'EOF'
fn main():
    print("Hello from Simple!")
EOF

# Run with interpreter
../../target/debug/simple main.spl

# Compile to binary
../../target/debug/simple compile main.spl -o my_app

# Celebrate! 🎉
```
