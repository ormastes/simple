## Description

Unordered collection of unique elements. Supports add, remove, contains, and set operations (union, intersection, difference).

## Operations

| Operation | Description | Complexity |
|-----------|-------------|------------|
| add | Add element | O(1) |
| remove | Remove element | O(1) |
| contains | Check membership | O(1) |
| union | Combine sets | O(n) |
| intersection | Common elements | O(n) |
| difference | Unique to first | O(n) |
