## Adding a New Language

### Step 1: Copy Existing Catalogs

Copy the English catalogs and rename with your language code:

```bash

cp i18n/__init__.spl i18n/__init__.ja.spl
cp i18n/parser.spl i18n/parser.ja.spl
cp i18n/compiler.spl i18n/compiler.ja.spl
```

## Message Structure

Each error message has these fields:

```simple
"E0001": {
    "id": "E0001",                    # Error code (DO NOT TRANSLATE)
    "title": "Error Title",           # Short title
    "message": "Full message {var}",  # Main message with placeholders
    "label": "Label text",            # Optional: code label
    "help": "Help text",              # Optional: suggestion
    "note": "Additional note"         # Optional: extra info
}
```

### Required Fields

- `id`: Error code (always keep in English)
- `title`: Short error title
- `message`: Main error message

### Optional Fields

Only include if present in the English version:
- `label`: Text for code labels
- `help`: Helpful suggestions
- `note`: Additional information
