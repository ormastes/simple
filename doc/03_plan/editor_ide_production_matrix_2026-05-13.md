# Editor IDE Production Matrix - 2026-05-13

Purpose: track production parity for the Markdown/wiki editor, VSCode-style
debugging, and LSP surfaces. A row is complete only when implementation, UI
wiring, and behavior verification exist.

## Status Legend

- PASS: behavior works end-to-end and is covered by tests.
- PARTIAL: some implementation exists, but parity is incomplete.
- MISSING: no production implementation was found.
- BLOCKED: depends on architecture/runtime support that is not present yet.
