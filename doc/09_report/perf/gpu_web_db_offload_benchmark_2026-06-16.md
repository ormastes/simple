# GPU Web/DB Offload Benchmark Report

This report is generated by `scripts/check/check-gpu-web-db-offload-benchmark-report.shs`.
It is host-safe benchmark-contract evidence: it proves metric shape, GPU-hit accounting,
CPU fallback accounting, backend timing provenance, tiny-batch rejection, a production-facing CPU-oracle web inference validation row, and optional measured CUDA DB/web contract evidence. Measured CUDA rows prove perf-only vector, columnar-scan, join/aggregate, NoSQL document-filter, web-inference, web-embedding, web-rank, and web-transform contract shapes; the validated web rows prove response replacement is gated by CPU/GPU candidate agreement and strict device-backend timing, and the SSD materialized scan rows prove a DBFS pager/WAL/checkpoint fixture can reach repeated strict DB profile device-validation boundaries using both contract and runtime queue timing. The standard-shape DB rows map the existing DB dispatch targets to ClickBench-inspired OLAP scan, TPC-H-style join/aggregate, BenchBase/YCSB document-filter, and ANN vector-search workload names while reporting unavailable external baselines. The combined production DB/web row aggregates the runtime-queue DBFS SSD scan and web-framework inference validations into one throughput contract; it is still a bounded contract row, not an external workload speedup claim.
